# Portfólio - João Marcos

Este é um projeto acadêmico desenvolvido para a disciplina de Aplicações Web e Móveis, do curso de Engenharia de Software da Uninter. A ideia foi montar um portfólio pessoal simples, usando apenas o básico de HTML, CSS e JavaScript, sem frameworks.

Página publicada: https://joao-m-csilva.github.io/portifolio-joaomarcos-uninter/

## Tecnologias usadas

- HTML
- CSS
- JavaScript

## Estrutura do projeto

```
portifolio-joaomarcos-uninter/
│
├── index.html              # Página principal, com todas as seções (início, sobre mim, formação, portfólio e contato)
│
├── src/
│   ├── styles/
│   │   ├── reset.css       # Reset básico de estilos do navegador
│   │   └── style.css       # Estilos do projeto (layout, grid do portfólio, cores, etc)
│   │
│   ├── scripts/
│   │   └── script.js       # Validação do formulário de contato
│   │
│   └── assets/              # Imagens usadas no portfólio
│
└── README.md
```

## Sobre o desenvolvimento

O site é dividido em seções acessadas por âncoras no menu (início, sobre mim, formação, portfólio e contato). As imagens do portfólio ficam organizadas em grid, cada uma com sua legenda. O formulário de contato tem uma validação simples feita em JavaScript, verificando se os campos foram preenchidos e se o e-mail tem um formato básico válido.
